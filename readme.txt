Adds registered CPT & custom taxonomy capabilities to Members plugin capabilities list

== Description ==

This is a very light-weight plugin that takes all capabilities associated with registered custom post types and custom taxonomies and adds them to the list of available capabilities in the [Members](http://wordpress.org/plugins/members) plugin by [Justin Tadlock](http://justintadlock.com.)

== Installation ==

1. Activate the plugin
